# 🖐️ Gesture Controlled Virtual Mouse

This project uses **Computer Vision** to control mouse actions using hand gestures through a webcam. It leverages **MediaPipe** for hand tracking and **PyAutoGUI** for performing mouse operations.

---

## 🚀 Features

* 🖱️ **Cursor Movement** – Move the mouse using your index finger
* 👆 **Left Click** – Perform left click using a specific finger gesture
* 👉 **Right Click** – Trigger right click gesture
* ✌️ **Double Click** – Use two-finger gesture for double click
* 📸 **Screenshot** – Capture screen using a hand gesture

---

## 🧠 Technologies Used

* Python
* OpenCV
* MediaPipe
* PyAutoGUI
* Pynput
* NumPy

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/gesture-mouse-control.git
   ```

2. Navigate to the project folder:

   ```bash
   cd gesture-mouse-control
   ```

3. Install dependencies:

   ```bash
   pip install mediapipe opencv-python pyautogui pynput numpy
   ```

4. Run the notebook:

   ```bash
   jupyter notebook main.ipynb
   ```

---

## 🎯 Project Idea

The system tracks hand landmarks in real-time and converts finger movements and gestures into mouse actions like clicking and screenshot capture.
